## 💡 使用技巧

### 在手机上访问

1. 在电脑浏览器打开前台

## 🔧 开发者说明

### CSS 媒体查询结构
```css
/* 默认样式（桌面） */
.element {
    /* 桌面样式 */
}

/* 平板和手机 */
@media (max-width: 768px) {
    .element {
        /* 平板样式 */
    }
}

/* 超小屏幕（手机） */
@media (max-width: 480px) {
    .element {
        /* 手机样式 */
    }
}
```

### 关键技术点
1. **Flexbox 方向切换**：`flex-direction: column`
2. **视口单位**：`width: 100vw`, `height: 100vh`
3. **触摸滚动**：`-webkit-overflow-scrolling: touch`
4. **文字缩放**：`font-size: calc(14px + 0.5vw)`
5. **防止缩放**：viewport 设置已优化
